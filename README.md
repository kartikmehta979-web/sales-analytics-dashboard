# 📊 Sales Analytics Dashboard

## 📌 Project Overview
This project analyzes retail sales data and presents business insights through an interactive Power BI dashboard.  
The goal is to understand revenue trends, regional performance, and product-level sales using data visualization.

This project demonstrates a real-world Data Analyst workflow:
- Data cleaning
- Data analysis
- Business insight generation
- Dashboard reporting

---

## 🎯 Business Objectives
- Measure overall sales performance
- Track monthly revenue trends
- Identify top-performing regions
- Analyze category-wise sales contribution
- Discover best-selling products

---

## 🛠 Tools & Technologies Used
- Power BI — Dashboard creation & data visualization
- Python (Pandas) — Data cleaning & preprocessing
- Excel / CSV — Data source handling

---

## 📂 Dataset Information
- Retail sales transaction dataset
- ~9,700 records
- Includes:
  - Order details
  - Customer information
  - Product categories
  - Regional data
  - Sales metrics

---

## 📈 Dashboard Features

### 💰 KPI Metrics
- Total Sales Revenue
- Total Orders
- Unique Customers
- Average Order Value

### 📊 Visual Reports
- Monthly Sales Trend
- Sales by Region
- Category Performance
- Top 10 Revenue-Generating Products

### 🧠 Insights Section
Key findings are highlighted directly on the dashboard for quick decision-making.

---

## 🔍 Key Business Insights
- West region generates the highest sales revenue
- Technology category contributes the largest share of revenue
- Canon ImageCLASS Copier is the best-selling product
- Sales show steady growth over time
